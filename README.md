# Sanic Forever
Sonic the Hedgehog demake...wip


Running the game
================
Recommended is a core 2 duo 2.4 ghz or higher


Default controls
================

w    => jump
down => crouch

- arrow keys move


Notes
=====
draw-physics-overlay drawing mode (sanicforever.ini) is very slow


Version Requirements
====================
pygame 1.9
pymunk 4
pymunktmx 0.7
pytmx 2.16.4
pyscroll 2.14.2
